Canvas demos

Tyto stránky obsahují pokusy s počítačovou grafikou využívající html5 canvas.
Jednotlivé skripty jsou založené na komponentách NextCanvas a NextGame, což je tenký wrapper pro canvas a jednoduchá třída obsluhující vstupy a game loop.
Zdrojové kódy viz github repository.
